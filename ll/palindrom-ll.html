Palindromic Linked List üß©

Given the head of a singly linked list, return true if it is a palindrome.

 

Approach 1: -

Insert all the elements into the vector, and then compare the elements. That is check whether the vector is palindromic or not.

 

bool isPalindrome(Node *head)

{

    Node *temp=head;

    vector<int> v;

    while(temp){

        v.push_back(temp->data);

        temp=temp->next;

    }

    int n=v.size();

    for(int i=0;i<(n/2);i++)

        if(v[i]!=v[n-i-1])

            return false;

           

    return true;

}

Time Complexity: O(n)

Space Complexity: O(n)

 

Approach 2: -

Take two pointers as slow and fast. Move till the fast pointer reaches to the second last element of the linked list. When the fast pointer approaches the second last element of the linked list, slow pointer will be at the mid of the linked list.

 

Now reverse the second half of the linked list.

 

Note that in the second case, the reverse(slow->next) would return 5 -> 4, which would be

Compared with the first two elements. Since the number of elements is odd here, we donÔøΩ need to compare the middle element. So only two comparisons will be performed here. :)

 

ListNode *reverse(ListNode *head){

    ListNode *curr=head;

    ListNode *prev=NULL;

    ListNode *nex=NULL;

    while(curr){

        nex=curr->next;

        curr->next=prev;

        prev=curr;

        curr=nex;

    }

    return prev;

}

bool isPalindrome(ListNode* head) {

    ListNode *fast=head;

    ListNode *slow=head;

    while(fast && fast->next && fast->next->next){

        fast=fast->next->next;

        slow=slow->next;

    }

    slow->next=reverse(slow->next);

    slow=slow->next;

    while(slow){

        // cout<<head->val<<" "<<slow->val<<endl;

        if(slow->val!=head->val)

            return 0;

        head=head->next;

        slow=slow->next;

    }

    return 1;

}

 

Time Complexity: O(n/2)

Space Complexity: O(1)
