Start of a Loop üîÅ

Given a Linked List with a loop, determine the node from where the loop has started :)

Algorithm:
1. Firstly find the collision node, using the above approach.

2. Once we find the collision node, we declare one more pointer as entry pointer, which will point to the starting node of the linked list.

3. Now we will move the entry pointer and the slow pointer simultaneously, until they collide.

4. The node, where the entry pointer and slow pointer collide will be the start of the node.

Why the entry pointer and the slow pointer are bound to meet at the start of the loop?

In this figure, let us assume that, the fast and slow pointer meet at node k.

Now the distance moved by slow and fast pointers are :
1. Slow = l1 + l2 2. Fast = l1 + nc + l2 (let's say that, the fast pointer traversed the loop n times)

Now since fast and slow pointer has met, and also the fast pointer move twice as fast as the slow pointer, thus the equation will be.
2 (l1 + l2) = l1 + l2 + nc
L1 + L2 = nc
L1 = (nc - L2) i.e. (a = b)
