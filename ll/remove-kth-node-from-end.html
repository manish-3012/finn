Remove kth Node from the End of a List üîö

Given a linked list and an integer N, the task is to delete the Nth node from the end of the given linked list.

Na√Øve Approach: -

We have to remove kth node from the end, so we will use the logic that kth node from the end will be (n-k)th from the beginning, where n is the total number of nodes. So we will firstly count the total number of nodes and then we will traverse till the (n-k) th node from the beginning and our answer will be (n-k)th node :)

Code: -

ListNode* removeNthFromEnd(ListNode* head, int n) {

    ListNode *t1=head, *t2=head;

    ListNode* prev=NULL;

    int c=0;

    while(t1!=NULL){

        t1=t1->next;

        c++;

    }

    // if n==c, then remove the first element

    if(c==n)

        return head->next;

    int k=c-n;

    // we will move forward t2 by k times,

    // so now t2 points to the node to be deleted

    while(k-- && t2){

        prev=t2;

        t2=t2->next;

    }

    prev->next=t2->next;

    t2->next=NULL;

    return head;

}

 

Efficient Approach (Using two Pointer) : -

Here we will make use of hare 🦌 and tortoise

Algorithm:

Step 1: We will keep two pointers at the head of the linked list, one as slow and another as fast.

Step 2: Firstly we will move the fast pointer i.e. hare k times so now it has only (n-k) nodes remaining in front of it.

Step 3: Now we will start moving slow and fast pointer ahead simultaneously, till the fast pointer reach the second last node of the linked list. When the fast pointer will reach the second last node of the linked list, at that time the slow pointer will be pointing to the (n-k-1)th node.

Step 4: So we have to remove the node next to the slow pointer, so simply point slow->next = slow->next->next, and return head :)

ListNode* removeNthFromEnd(ListNode* head, int k){

    ListNode *slow=head, *fast=head;

    while(k--)

        fast=fast->next;

    if(fast==NULL)

        return slow->next; remov ing the first element :)

    while(fast && fast->next){

        fast=fast->next;

        slow=slow->next;

    }

    slow->next=slow->next->next;

    return head;

}

 

// move fast pointer k steps ahead. so now it will move

// (n-k) steps ahead to reach the secind last pointer

// so the slow poi ill also move (n- k) step s

Note: at the end, the fast pointer will be pointing to the second last node of the, linked list. So the slow pointer will be pointing to the previous node of the node to be deleted.
