Add two numbers represented as Linked Lists ‚ûï

Given two numbers represented by two lists, write a function that returns the sum list. The sum list is a list representation of the addition of two input numbers.

Intuition: Keep track of the carry using a variable and simulate digits-by-digits sum starting from the head of the list, which contains the least significant digit.

 

Approach: 

342 + 465 = 807

342+465=807.

Each node contains a single digit and the digits are stored in reverse order.

Just like how you would sum two numbers on a piece of paper, we begin by summing the least significant digits, which is the head of l1 and l2. Since each digit is in the range of 0‚Ä¶9, summing two digits may ‚Äúoverflow‚Äù. For exam bsp;

5 + 7= 12. In this case, we set the current digit to 2 (12%10) and bring over the carry=1 (12/10) to the next iteration. 

carry must be either 0 or 1 because the largest possible sum of two digits (including the carry) is 9 + 9 + 1 = 19.

 

Psuedocode:

1. Create a dummy node which is the head of new linked list.

2. Create a node temp, initialise it with dummy.

3. Initialize carry to 0.

4. Loop through lists l1 and l2 until you reach both ends, and until carry is present.

      a. Set sum=l1.val+ l2.val + carry.

      b. Update carry=sum/10.

      c. Create a new node with the digit value of (sum%10) and set it to temp node‚Äôs next, then

      d. advance temp node to next.

      e. Advance both l1 and l2.

5. Return dummy‚Äôs next node. 

Note that we use a dummy head to simplify the code. Without a dummy head, you would have to write extra conditional statements to initialize the head‚Äôs value.Take extra caution in the following cases:

 

Test case

Explanation

l1=[0,1], l2=[0,1,2]

When one list is longer than the other.

l1=[], l2=[0,1]

When one list is null, which means an empty list.

l1=[9,9], l2=[1]

The sum could have an extra carry of one at the end, which is easy to forget.

ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {

    ListNode* ans=new ListNode();           //to return as the final ans

    ListNode* temp=ans;                  //to keep the nodes in the ans linked list

    int carry =0;

    while(l1 || l2 || carry){

        int s=0;

        if(l1){

            s+=l1->val;

            l1=l1->next;

        }

        if(l2){

            s+=l2->val;

            l2=l2->next;

        }

        s+=carry;

        carry=s/10;

        ListNode *node=new ListNode(s%10);

        temp->next=node;

        temp=temp->next;

    }

    return ans->next;

}

 

There exist another solution as well, which is simply extracting the numbers from the linked list, adding them and then putting them back into a linked list, but this is not going to work as we can't add/store very large inputs.
