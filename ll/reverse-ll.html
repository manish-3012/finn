Reverse a Linked List üîÄ

Given a Linked List, reverse all the node pointers such that the end of the linked list becomes the first element and vice versa.

In short we could say that for any particular pointer we will keep a track of previous and next pointer along with the current pointer which will be helping us in the traversal of the linked list :)

So firstly we will keep a track of the next pointer of the current pointer as we are going to break the link after the current pointer. For the current pointer we will simply point its next to the previous pointer and then make current pointer equal to the next pointer.

Algorithm:
1. We will declare three pointers, namely *curr, *prev and *nex, where *curr will help us in traversing through the linked list.

2. While *prev will keep a track of previous of the current and *nex will keep a track of the next pointer of the *curr.

3. Point *nex to the curr->next as we will be breaking the link from the *curr pointer onwards.

4. Now break the link and point curr->next to the previous pointer i.e. *prev.

5. Now we will simply move forward *curr and *prev.

6. Repeat steps 3-6 till current does NOT become equal to NULL.
