Make Beautiful Matrix üîÆ

A matrix is called special if the sum of each row and the sum of each column is the same.

Given a square matrix AA of dimensions N√óNN√óN, find the minimum number of operations required to make it special. In one operation you can choose any number in the matrix and increase it by 1.

We will simply calculate, the maximum sum possible from all the rows as well as column. We will also find the total sum of all the elements of the given matrix, from the maximum possible sum, from all the row and column we will calculate the final ans.

Algorithm:
	1. Our first task is to find out the row/column with the maximum sum. Since all the other rows and columns will have to have that same maximum sum only for the matrix to be a special.
	2. To find out the row/column with the maximum sum, we will traverse through the matrix and find out the sum of row and column elements, 
	NOTE: for any particular i and j, matrix[i][j] will give us the summation of row while matrix[j][i] will give us the summation of column :)
	3. Finally we will take the maximum among the maxRow and maxCol, and then return (maxrc*n) - s, where maxrc is the overall maximum among all the rows and columns, while s is the sum of all elements of the array :)
