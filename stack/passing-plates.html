Passing the Plates üçΩÔ∏è

We are passing a sequence of plates from Atul to Zenobia. Each plate has a number painted on it and no two plates have the same number. We have a table in front of us on which we can temporarily store a single stack of plates. At each step, we are allowed to do one of the following:

‚Ä¢ Take a plate from Atul and pass it on imme iately to Zenobia.

‚Ä¢ Take a plate from Atul and put it on top o the stack on the table.

‚Ä¢ If there is at least one plate in the stac on the table, take the topmost plate off the stack and pass it on to Zenobia.

 

In this process, we can rearrange the plates that Atul gives us before passing them on to Zenobia. For instance, if the sequence of numbers on the plates given to us by Atul is 1,2,3,4, we can pass them onto Zenobia in the sequence 2,4,3,1 as follows:

 

‚Ä¢ Take plate 1 from Atul and start a stack o the table.

‚Ä¢ Pass plate 2 directly from Atul to Zenobia

‚Ä¢ Take plate 3 from Atul and stack it on top of plate 1.

‚Ä¢ Pass plate 4 directly from Atul to Zenobia

‚Ä¢ Take plate 3 off the stack and pass it to enobia.

‚Ä¢ Take plate 1 off the stack and pass it to enobia.

 

Algorithm: -

1. Make two pointers i and j, pointing to the arr and the req.

2. Compare arr[i] == req[j], if so then we are directly passing the plates from Atul to Zen, so just increasing both the pointers

3. If they are not equal, then check whether the req[j] == st.top(), if yes then we will be passing the plate from the stack to the req[], so pop from the stack, and increase j.

4. If the req[j] != st.top(), then we will have no option, but to push the plate on the stack.

5. Once all the elements of the arr are traversed, i.e. i==n, then start popping from the stack, till j=n,

6. At the end check if, stack is empty as well as j=n, if yes then the req[] array is valid else not.

 

Intuition: The logic behind this algorithm is that, we will keep traversing the array with i while if the arr[i] is equal to the req[j] then we will simply pass it to Zen and increase j, since we got one more element from the required array. While if it is not equal then we will check if the top of the stack is equal to the req[j], if yes then we will pass the top plate from the stack to Zen. And if both of the above cases doesn't occur then we have only one option and that is to push the plate from Atul into the stack.

 

bool solve(int a[], int b[], int n){

    stack<int> st;

    int i=0, j=0;

    while(i<n){

        if(a[i]==b[j]){

            i++; j++;           //directly passing

        }

        else{

            if(st.size() && st.top()==b[j]){

                j++;        //passing from stack

                st.pop();

            }

            else{

                st.push(a[i++]);   //pushing in the stack

            }

        }

    }

    while(st.size() && st.top()==b[j]){

        st.pop();

        j++;

    }

    if(st.size()==0 && j==n)

        return true;

    return false;

}

 

Complexity Analysis:

Time: O(N): We need to check only each number in the sequence.

Space: O(N): We need to maintain an extra stack, this stack can contain the entire

array in the worst case.
