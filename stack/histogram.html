Maximum Area Histogram üìö

Find the largest rectangular area possible in a given histogram where the largest rectangle can be made of a number of contiguous bars. For simplicity, assume that all bars have the same width and the width is 1 unit, there will be N bars height of each bar will be given by the array arr.

Example 1:

Input:
N = 7
arr[] = {6,2,5,4,5,1,6}
Output: 12

A building can be expanded into other buildings only when, the other buildings have height greater than or equal to the current building.

 

int solve(vector<int> arr){

    int n=arr.size();

    int ans=0;

    vector<int> vl, vr;

    vl=nsl(arr);

    vr=nsr(arr);

    for(int i=0;i<n;i++){

        int w = vr[i] - vl[i] -1;

        ans = max(ans, w*arr[i]);

    }

    return ans;

}

 

Time Complexity: O(n2)

Space Complexity: O(2n)

 

Where nsl and nsr are determined as shown in the NGR, NSR, NGL, NSL

long long getMaxArea(long long v[], int n){

    vector<long long> r(n,n), l(n,-1), w(n);

    stack<pair<long long,long long>> s1, s2;

    s1.push({0,v[0]});

    for(int i=1;i<n;i++){  √Ç √Ç √Ç √Ç ÔøΩ //to find the nearest minimum to right

        while(s1.size() && s1.top().second>v[i]){

            r[s1.top().first]=i;

            s1.pop();

        }

        s1.push({i, v[i]});

    }

    s2.push({n-1,v[n-1]});

    for(int i=n-1;i>=0;i--){ //to find the near nimum to left

        while(s2.size() && s2.top().second>v[i]){

            l[s2.top().first]=i;

            s2.pop();

        }

        s2.push({i, v[i]});

    }

    long long ans=v[0];

    for(int i=0;i<n;i++){

        w[i]= r[i] - l[i] -1;

        ans=max(ans, v[i]*w[i]);

    }

    return ans;

}

 

Time Complexity: O(n2)

Space Complexity: O(2n)
