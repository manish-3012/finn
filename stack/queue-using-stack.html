Queue using Stack üëΩ

Implement a queue using two stacks and perform the following type of queries.

1 X: enqueue x

2: dequeue

3: print the front element of queue.

 

Algorithm:

1. We will maintain two stacks, one would act as the rear of the queue, i.e. we are going to push in that stack during enqueue operation

2. The other stack will act as the front of the queue and we will perform the dequeue as well as the print operation using that stack.

3. During all the enqueue operations simply push the elements in the rear stack.

4. If it‚Äôs dequeue operation and the front stack is empty, simply push all the elements from the rear to the front.

5. While if the front stack is not empty we will simply pop the top of the front.

Similarly, during print we will simply print the top element of the front stack.

 

For simplicity take it as, the two stacks that is back and front are internally connected, such that the front element is the front part, and the rear is the end part.

stack<int> rear, front;

void enqueue(int x) {

    rear.push(x);

}

int dequeue() {

    if(front.size()){

        int x= front.top();

        front.pop();

        return x;

    }

    else{

        while(rear.size()){

            front.push(rear.top());

            rear.pop();

        }

        int x= front.top();

        front.pop();

        return x;

    }

}

 

Time Complexity: O(n)

Space Complexity: O(2n)
