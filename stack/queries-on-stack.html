Queries on Stack üëΩ

You have an empty sequence, and you will be given N queries. Each query is one of these three types:

1-Store the element x into an array.

2 -Delete the most recently stored element.

3-Print the maximum element in the array.

 

The first line of input contains an integer,N. The next N lines each contain an above mentioned query. (It is guaranteed that each query is valid). For each type query, print the maximum element in the stack on a new line.

 

Algorithm:

1. We will have to make two stack here one to store the elements and another for keeping the maximum element present in the actual stack,

3. For the second purpose, it's easier to use a priority_queue, which will store the maximum element present in the stack at the top.

4. If q=1, simply push the element in the stack and the priority queue

5. If q=2, pop the element from the stack, but before popping check if the top of the stack is same as the top of the priority queue, in that case, we have to remove the maximum element only, so remove from the priority queue as well.

6. If q=3, simply print the top of the priority queue.

 

Code:

while(t--){

        int a;

        cin>>a;

        if(a==1){

            int e;

            cin>>e;

            pq.push(e);

            st.push(e);

        }

        else if(a==2){

            if(st.size()){

                if(st.top()==pq.top())

                    pq.pop();

                st.pop();

            }

        }

        else{

            cout<<pq.top()<<endl;

        }

    }

 

Complexity Analysis:

Time: O(N): We need to check only each number in the sequence.

Space: O(N): We need to maintain an extra stack, this stack can contain the entire array in the worst case.
