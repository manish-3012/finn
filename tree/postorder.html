1.2 Pre-order traversal: - (Root, Left, Right)

 

Algorithm:

1) Visit the root

2) Traverse the left sub-tree (in-order left sub-tree)

3) Traverse the right sub-tree (in-order right sub-tree)

Application:

1) Pre-order traversal is used to create a copy of the tree. 

2) Pre-order traversal is also used to get prefix expression on an expression tree.

 

Code:

void util(Node *root, vector<int> &v){

    if(root!=NULL){

        v.push_back(root->data);

        util(root->left,v);

        util(root->right,v);

    }

}

vector<int> preorder(Node* root) {

    vector<int> ans;

    util(root, ans);

    return ans;

}

 

Time Complexity: O(n)

Iterative Approach: -

 

Algorithm:

1) Create an empty stack nodeStack and push root node to stack. 

2) Do the following while nodeStack is not empty. 

a) Pop an item from the stack and print it. 

b) Push right child of a popped item to stack 

c) Push left child of a popped item to stack

 

The right child is pushed before the left child to make sure that the left subtree is processed first.

 

vector<int> preorder(Node* root) {

    vector<int> ans;

    if(!root)

        return ans;

    stack<Node *> st;

    st.push(root);

    while(!st.empty()){

        Node *temp=st.top();

        v.push_back(temp->data);

        st.pop();

        if(temp->right)

            st.push(temp->right);

        if(temp->left)

            st.push(temp->left);

    }

    return ans;

}

 

Time Complexity: O(n) 

 

Space Complexity: T(n)
