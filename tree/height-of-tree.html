Code:

int height(struct Node* root){

   if(root==NULL)

      return 0;

   int lh=height(root->left);

   int rh=height(root->right);

   int ans= 1+ max(lh,rh);

   return ans;

}

 

Time Complexity: O(n)

 

Using level Order Traversal:

We could find the height of a binary tree using level order traversal as well, what we do is to push NULL in the queue, after all the elements of a particular level are pushed into the queue/

Whenever we encounter a NULL, it indicates the end of a level, so we increase the height by one. For more details check out Level Order Traversal

 

Algorithm:

1) Push the root into the queue, and also push NULL, to indicate level 1 being pushed

2) Now iterate through the queue, till it becomes empty

3) If we encounter a NULL node, it indicates the end of a level, so we increment our ans by 1

4) Else push the curr->left as well as right (if not null)

5) At the end of loop part, check if the queue became empty, if not empty then push NULL

 

Funda: Think as if, all the nodes of a particular level are closed by a NULL node, and the number of such NULL nodes helps us in determining the height of a tree

 

int height(struct Node* root){

   if(root==NULL)

      return 0;

   int ans=0;

   queue<Node *> q;

   q.push(root);

   q.push(NULL);  //to indicate level 1

   while(!q.empty()){

      Node *temp=q.front();

      q.pop();

      if(temp==NULL) //we encountered end of a level

         ans++;

     

      if(temp!=NULL){  //nodes are yet to be traversed

         if(temp->left)

            q.push(temp->left);

         if(temp->right)

            q.push(temp->right);

      }

      else if(!q.empty())  //if queue still has elements

         q.push(NULL);

   }

   return ans;

}

 

Time Complexity: O(n)

Space Complexity: O(n)
